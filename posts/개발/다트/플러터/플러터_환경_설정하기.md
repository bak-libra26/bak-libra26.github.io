# M1 Mac에 다트/플러터 설치 방법

> 본 글에서는 M1 Mac 환경에 Homebrew를 사용하여 다트/플러터를 설치하는 방법을 설명합니다.

## Homebrew 설치

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## 다트/플러터 설치

```bash
brew install dart
```
