젠킨스 JDK

  • 젠킨스 빌드용 JDK 필요함
  • 로그인 → Manage Jenkins → Tools → JDK installations 이동
  • Add JDK 클릭, Name(별칭), JAVA_HOME(설치 경로) 입력
