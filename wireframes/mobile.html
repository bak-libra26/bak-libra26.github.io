Overview

JWT(JSON Web Token)는 당사자 간에 정보를 안전하게 전달하기 위한 토큰 기반 인증 방식입니다.

세션 기반 인증과 달리 서버가 상태를 저장하지 않으므로, 수평 확장이 용이합니다.

구현

@Component
public class JwtTokenProvider {
    public String generateToken() {
        return Jwts.builder()
            .setSubject(auth.getName())
            .compact();
    }
}
